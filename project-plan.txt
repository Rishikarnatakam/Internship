Vite React TypeScript project to reproduce Midday dashboard UI from screenshots.

Steps to run locally:
1. npm create vite@latest midday-dashboard -- --template react-ts
2. cd midday-dashboard
3. npm install
4. npm install tailwindcss postcss autoprefixer recharts framer-motion clsx axios
5. npm install @ai-sdk-tools/store @ai-sdk-tools/artifacts @ai-sdk-tools/devtools
6. npx tailwindcss init -p
